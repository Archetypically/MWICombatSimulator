# Typography

- Geist for Body Text
- Geist Mono for Codeblocks
- Geist Pixel for Layout elements (headers, heros, nav, footer, etc).

# Component Library

- Use [shadcn/ui](https://ui.shadcn.com/) for base components.

```bash
pnpm dlx shadcn@latest add button
```

# Icons

- Use [lucide-animated](https://lucide-animated.com/) for icons.
- Important: Do not use the default lucide icons. Always use the animated versions.

```bash
pnpm dlx shadcn add @lucide-animated/icon-name
```
